# Foto Kita Mulai Blur

Sebuah website modern untuk pengalaman kamera live dengan gesture peace yang otomatis menerapkan efek blur. Situs ini dibuat dengan Next.js, MediaPipe hand tracking, dan YouTube background audio.

## Fitur

- Splash screen estetis dengan logo kustom
- Tombol start dan transisi halaman yang halus
- Kamera permission explicit, hanya aktif saat pengguna memilih
- Deteksi gesture peace untuk menerapkan blur
- Kontrol audio tersembunyi: play, pause, stop, dan volume
- SEO metadata dan favicon siap produksi

## Teknologi

- Next.js App Router
- Tailwind CSS
- `@mediapipe/tasks-vision`
- `framer-motion`
- `react-youtube`

## Menjalankan lokal

```bash
npm install
npm run dev
```

Buka `http://localhost:3000` untuk melihat aplikasi.
